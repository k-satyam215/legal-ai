# ⚖️ AI Legal Advisor (India)

An AI-powered legal assistant that provides:
- Legal guidance based on Indian laws
- Step-by-step actions
- Automated legal notice generation (PDF)

## 🚀 Features
- RAG (Supreme Court Judgments)
- LLaMA 3.3 70B (Groq)
- Streamlit UI
- Dynamic Legal Notice Generator

## 🛠️ Tech Stack
- Python
- LangChain
- FAISS
- Streamlit
- Groq API

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
